# piano-role-movie2music-score
YouTubeなどに存在しているピアノロール動画使用して、楽譜の復元を行います。
本来ピアノロール動画はDTMソフトウェア上で作成したmidiファイルを使用して作成されるものですが、本プロジェクトではその逆を行うことで、音楽データの抽出を試みます。

# 更新
2023/04/12:動画のピアノロールの場所を指定すると、自動解析をし音符の抽出に成功
